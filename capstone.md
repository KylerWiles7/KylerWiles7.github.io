### SceneManager.cpp
SceneManager.cpp is the source code for the scene in my OpenGL project. SceneManger.cpp utilizes the textures, scene lights, objects, shaders, transformations and locations. I originally worked on this code in October 2025 at SNHU.

I chose this artifact for my ePortfolio because it showcases my ability to work with older software, the skills I used to enhance C++ code, my ability to enhance previous work and my understanding of complex source code. OpenGL was created in 1992 and most of the software systems used today are even older. This artifact shows that I am capable of using older software and also capable of adding meaningful enhancements that improve the base code. The code in this artifact is written in C++ and I was able to integrate a fully visualized scene using only C++ code. This showcases my skill with C++ and my ability to improve the project by fully visualizing the scene. This artifact also shows that I am able to revisit older projects and add meaningful additions to the original code. I think this is important if there is a need for you to back-track to a previous project. SceneManager.cpp is a large and complex source file, which is expected because it manages the entire scene. The size and complexity demonstrates my undestanding to navigate and work with complex source code.
